# Blind Vision Guide – Object Detection with Voice Navigation

A Python desktop application that helps visually impaired users detect nearby objects and receive real-time voice navigation in **English** and **Tamil**.

## ✨ Features

✅ **Real-time Object Detection** - Detects objects using your webcam  
✅ **Voice Output** - Speaks detected objects aloud (English & Tamil)  
✅ **Easy GUI** - Simple button-based control interface  
✅ **Obstacle Warnings** - Alerts about potential obstacles  
✅ **Bilingual Support** - Works in both English and Tamil  

## 🛠️ Tech Stack

- **Python 3.9+**
- **OpenCV** - Webcam capture and image processing
- **TensorFlow** - Pre-trained MobileNetV2 for object detection
- **pyttsx3** - English text-to-speech
- **gTTS** - Google Text-to-Speech for Tamil
- **Tkinter** - Desktop GUI framework
